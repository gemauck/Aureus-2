Troubleshooting

If you see "Loading Pipeline..." or blank screen:
  1. Open browser console (F12)
  2. Look for any red error messages
  3. Type: console.log(window.Pipeline)
  4. Should show: function Pipeline()
If Pipeline menu item doesn't appear:
  1. Make sure you saved MainLayout.jsx changes
  2. Hard refresh browser (Ctrl+Shift+R or Cmd+Shift+R)
  3. Clear browser cache
If no deals show:
  1. Go to Clients module first
  2. Add some test leads
  3. Return to Pipeline - they should appear
