# ERP System Upload Instructions

## Step 1: Create package.json
1. Go to: https://github.com/gemauck/abcotronics-erp
2. Click "Add file" → "Create new file"
3. Name: package.json
4. Copy this content:

{
  "name": "abcotronics-erp-modular",
  "private": true,
  "version": "0.1.0",
  "type": "module",
  "scripts": {
    "dev": "vercel dev",
    "build": "echo 'Frontend builds via static index.html; API is serverless'",
    "vercel-build": "prisma generate && prisma migrate deploy",
    "postinstall": "prisma generate",
    "prisma:generate": "prisma generate",
    "prisma:migrate": "prisma migrate dev",
    "prisma:deploy": "prisma migrate deploy",
    "seed": "node prisma/seed.js",
    "start:local": "node server/local.js"
  },
  "dependencies": {
    "@prisma/client": "^5.18.0",
    "bcryptjs": "^2.4.3",
    "express": "^4.19.2",
    "cors": "^2.8.5",
    "helmet": "^7.1.0",
    "jsonwebtoken": "^9.0.2",
    "pino": "^9.3.2",
    "zod": "^3.23.8"
  },
  "devDependencies": {
    "prisma": "^5.18.0"
  }
}

5. Commit: "Add package.json"

## Step 2: Create vercel.json
1. Click "Add file" → "Create new file"
2. Name: vercel.json
3. Copy this content:

{
  "functions": {
    "api/**/*.js": {
      "runtime": "nodejs18.x"
    }
  },
  "rewrites": [
    {
      "source": "/api/(.*)",
      "destination": "/api/$1"
    }
  ]
}

4. Commit: "Add vercel.json"

## Step 3: Create .gitignore
1. Click "Add file" → "Create new file"
2. Name: .gitignore
3. Copy this content:

node_modules/
.env
.env.local
.env.development.local
.env.test.local
.env.production.local
.DS_Store
*.log
dist/
build/

4. Commit: "Add .gitignore"

## Step 4: Create README.md
1. Click "Add file" → "Create new file"
2. Name: README.md
3. Copy this content:

# Abcotronics ERP System

A comprehensive ERP system for fuel management services.

## Features
- Client Management
- Project Tracking
- Invoice Management
- User Authentication
- Database Integration

## Setup
1. Install dependencies: npm install
2. Set up environment variables
3. Run database migrations
4. Start development server

4. Commit: "Add README.md"

## Next Steps
After uploading these 4 files, we can add the folders:
- api/ (API endpoints)
- prisma/ (Database schema)
- src/ (React components)
