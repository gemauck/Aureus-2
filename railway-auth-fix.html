Manual Fix Instructions

If the automated tests fail, follow these steps:

  1. Check Railway Environment Variables:
    railway variables
  2. Set Database URL:
    railway variables set DATABASE_URL="your_postgres_connection_string"
  3. Set JWT Secret:
    railway variables set JWT_SECRET="your-super-secret-jwt-key-change-this-in-production"
  4. Redeploy:
    railway up

Alternative: Use Local Development

For local development, you can:

  1. Set up a local PostgreSQL database
  2. Update your .env file with local DATABASE_URL
  3. Run: npm run seed
  4. Start local server: npm run dev
